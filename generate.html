Problem: Coding without the coding.

Generate is a learning tool for novice programmers that can translate pseudocode or written descriptions of code into functional code. Essentially a natural language compiler, Generate works by employing the fundamental principles of Natural Language Processing, and machine learning to parse and classify your input. In order to do this and ensure that users are getting the correct program, supervised learning is used. Furthermore, for each potential command, Generate requires the corresponding template, which needs to be tagged itself. That is why at this point in time, being a relatively new project, the number of user commands generate can interpret is rather limited.

Development

In all honesty, I came up with the idea after when I wanted a BASH script to help me check for differences between a series of files, but it was 4am and I didn't want to have to write and test a script to help me debug code.
I have always wanted to learn about NLP, since I first discovered Cleverbot back in elementary. It's taken me some time to get to a point where I felt confident enough to pursue this passion instead of waiting around for a class to teach it to me.

Ultimate Goal

My ultimate goal is to transform this application into an effective learning tool for novice programmers. In order to do that I need to be able to develop this program to be able to handle a variety of commands in the English language.

Current Step

To prepare this site for showcasing, I have trained the classifier to recognize the pseudocode command for a for loops and for addition with assignment.Presently, when I receive a command I will split the sentence and parse each word into 3-grams. I then group common words into the basic commands I have seen frequently used throughout my computer science career.
ie. Loop, if, sort, find, merge, add, etc.
